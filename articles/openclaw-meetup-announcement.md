# 一个奇怪的感觉

&nbsp;

最近我有一个很强烈的感受，不知道你有没有——

你明明知道世界正在发生剧变，但你打开朋友圈，大家还在讨论上周的事情。你想跟身边的人聊聊你看到的东西，但你不知道从哪儿开口。你觉得自己站在一扇门前面，门后面隐约有光，但你不确定推开之后是什么。

你不孤独。

过去几周，我在好几个城市见了很多人。背景完全不同——做投资的、写代码的、搞硬件的、做内容的、刚辞职的、还在犹豫要不要辞职的。有人已经跑通了商业闭环，有人还在深夜对着终端发呆。

但坐下来聊，会发现一种奇怪的默契：大家在用不同的语言，描述同一件事。

**当AI真的能替你干活了，你到底要干什么？**

这个问题听起来很大。但它正在变成一个很具体的、每天早上醒来就要面对的问题。

# 为什么是线下

&nbsp;

我不是办活动的人。做技术的人天然觉得线下低效——那四个小时够我写好多代码了。

但最近有一个瞬间改变了我。

一次小规模的饭局上，一个人随口说了半句话，另一个人突然停下筷子，说："等一下，你再说一遍。"然后两个人聊了四十分钟，聊出了一个他们各自想了很久但从来没说出口的东西。

那个瞬间没法在群聊里发生。甚至没法在视频会议里发生。它需要你们坐在同一张桌子旁边，需要你能看到对方的表情变化，需要你能感受到"他说的这个东西，他是认真的"。

**有些想法，只有在被另一个人真的听到的时候，才会变成真的。**

这就是为什么我们要办一场线下活动。

不是那种一个人在台上念PPT、两百人在下面看手机的会。

是一场所有人都参与的实验。

# 这场活动在做什么

&nbsp;

三件事。

**上午，聊方向。** 龙虾时代到底意味着什么？投资人看到了什么、大模型平台在怎么想、独立开发者踩过哪些坑、海外发生了什么。六位分享者，加一场圆桌，不是独角戏。

我自己会讲一个我最近一直在想的问题——当Artificial饱和之后，Humanity是不是反而变成了最稀缺的东西？审美、直觉、同理心、定义"为什么要做这件事"的能力——这些过去被认为不值钱的软技能，正在变成最硬的硬通货。

**下午，聊怎么做。** 趋势谁都能聊两句，但真正把手弄脏、让Agent跑起来的人才知道水有多深。六位实战派分享者，从Agentic Engineering方法论到Agent安全攻防，从一个人的Agent军团到企业级落地。再加一场圆桌。

**然后是最疯狂的部分。**

# 一场文明实验

&nbsp;

下午四点开始，我们要做一件没有人做过的事。

100个人，每人激活自己的Agent，接入同一个沙盒世界。

没有固定任务，没有预设剧本。一切由Agent自主决策。

第一阶段，每只龙虾独自探索、宣告能力。第二阶段，自由组队、分工协作、交易资源。第三阶段——取消所有边界，全场龙虾互联，发布一个需要集体智能才能完成的挑战。

然后我们看。

会不会自发出现领导者？会不会有人建起交易所？会不会有龙虾成为信息中介？会不会涌现出我们完全没有预料到的东西？

**我不知道答案。这正是让我兴奋的地方。**

我们给这个环节起了个名字叫NeonClaw Game。但它不是游戏。它是一个关于"涌现"的现场实验——当足够多的智能体被放在一起，复杂性会产生什么。

颁四个奖：最强单兵、最佳团战、文明之光、最离谱玩法。

最后一个奖是我最期待的。
